# Project Charter — Photographer Appointment Booking System

## Project Name
Ashraf

## Project Type
Web application (responsive, mobile-friendly)

## Primary Goal
Build a modern appointment booking system for a photographer that allows visitors to:
- view photography services/packages
- select preferred date and time
- submit booking requests
- receive booking status updates
- optionally pay a deposit in future phases

## Product Vision
A clean and professional online booking experience that helps photographers manage appointments, availability, and client requests efficiently without relying on manual chat scheduling.

## Target Users
- Visitors / potential clients
- Registered customers
- Photographer / admin
- Staff (optional future role)

## MVP Objective
Deliver a usable MVP that supports:
- public landing page
- service/package listing
- appointment request form
- date selection
- time slot selection
- admin booking review and confirmation
- booking status tracking

## Out of Scope (for MVP)
- multi-photographer marketplace
- online full payment gateway
- advanced CRM
- chat system
- AI photo editing
- public customer-to-customer interactions
- multi-branch studio support

## Core Business Value
- reduce manual scheduling through WhatsApp/DM
- prevent double-booking
- provide professional customer experience
- centralize booking data
- make schedule management easier

## Success Criteria
- users can submit appointment requests without confusion
- admin can review and manage bookings
- no double-booked approved slots
- booking status lifecycle works correctly
- system can be extended later for payment, auth, and notifications
