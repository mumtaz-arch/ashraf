# Feature Breakdown

## Phase 1 — MVP Core

### F1. Landing Page
- hero section
- brief introduction
- CTA to book now
- featured services
- trust section
- contact section

### F2. Services / Packages
- list of photography packages
- package name
- short description
- price display
- duration display
- "Book Now" action

### F3. Booking Form
- customer name
- phone / WhatsApp
- email (optional or required based on business rule)
- selected service
- event/session note
- preferred date
- preferred time slot
- optional special request

### F4. Slot Availability
- date-based available slots
- hidden unavailable slots
- blocked slots not selectable
- prevent duplicate confirmed bookings

### F5. Booking Submission
- validate form
- validate slot
- create booking
- default status = pending
- generate booking reference code

### F6. Admin Booking Management
- list bookings
- filter by status
- open booking detail
- update status
- add internal note

### F7. Basic Availability Management
- create available slot
- edit slot
- block slot
- remove slot
